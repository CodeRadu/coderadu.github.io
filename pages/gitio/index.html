Creating a repository
Create a repository with the name username.github.io
Getting something to show up

Open a folder and a terminal in that folder.
Make sure you have git installed.
Run git init, git branch -M main
Make an index.html file and type something in it, then run git add .
Run git remote add origin https://github.com/username/username.github.io.git
Then to push it run git push -u origin main
It's done!

Go to username.github.io and admire your work
